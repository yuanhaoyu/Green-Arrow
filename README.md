# Green-Arrow
---

<img src="./logo.png"  width="125">

Green Arrow 是一款集打点，监控为一体的系统。

## 主要功能：

### Arrow（箭）：

目前提供三种Arrow作为标记：
	
    
#### action-arrow:
    action-arrow 为动作箭。在我们需要记录的动作的时候会触发的action-arrow

    
#### star-arrow:
	
	star-arrow 为星箭，可以追踪页面元素的曝光情况
		
### watcher （守望者）：

守望者目前提供2种类型，错误守望者和接口守望者
	
	
#### error-watcher：
		
	目前error-watcher 有主动和被动2种方式
	
#### api-watcher:
	
	目前api-watcher基于配置实现
